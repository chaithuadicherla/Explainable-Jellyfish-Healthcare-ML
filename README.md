# Explainable Jellyfish Optimization Framework for Interpretable Machine Learning in Healthcare

## Overview
This project develops a healthcare machine learning framework for breast cancer prediction using the Jellyfish Search Algorithm (JSA) for feature selection and optimization.

## Key Features
- Random Forest classifier
- Jellyfish Search Algorithm (JSA)
- Multi-objective optimization
- SHAP explainability
- Comparison with GA and PSO

## Results
- Accuracy: 97.37%
- Feature Reduction: 73.33%
- Strong interpretability using SHAP

## Dataset
Breast Cancer Wisconsin Dataset (Scikit-learn)

## Technologies Used
- Python
- Scikit-learn
- Pandas
- NumPy
- Matplotlib
- SHAP
